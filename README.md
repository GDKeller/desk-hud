# Desk H.U.D.
This is a Heads Up Display for a workstation, acting as a companion to provide relevant information.

Designed to run on a Raspberry Pi 4 with 3.5" screen.



### Planned Features:

- [x] Clock
- [ ] Real-time weather
- [ ] Workstation stats
    - [ ] CPU usage
    - [ ] GPU usage
    - [ ] Memory usage
    - [ ] Fan speed
- [ ] Reminders
- [ ] Multiple "dashboard" screens to switch between
